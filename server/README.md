# ClothSwap Backend

This is the Express backend for the Clothing Swap Marketplace.

## 🛠️ Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)

## 🚦 Getting Started

1. `npm install`
2. Create a `.env` file based on `.env.example`.
3. `npm run dev` (uses nodemon) or `npm start`

## 📡 API Endpoints
- `/api/auth` - Authentication
- `/api/clothes` - Clothing items
- `/api/swaps` - Swap requests
- `/api/courier` - Shipping info
- `/api/stats` - Dashboard data
